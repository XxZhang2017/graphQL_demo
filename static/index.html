GraphQL AnalyzeURL

    Description:

  1. - success “OK”, “Not OK”
  2. - return code (e.g 200 if it was successful, 404 if the URL was invalid / page not found…)
  3. - size of the page hosted at that url
  4. - number of images
  5. - size of the page hosted at that url
  6. - and for each image referenced at that URL the URL of the image and their size and wether the URL was valid
  7. - support urls that are either http or https, and return errors if the url is invalid

Please go to /graphql to use graphial interface to see the demo. You can copy the query below entering the url within the quotation and make the query.

  query {     analyzeUrl ( url: " ") {

             success

             returnCode

             size

             numberOfImages

              images {

               url

               size }

               valid }

             }}
